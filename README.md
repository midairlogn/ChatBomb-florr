# ChatBomb-florr
这是`florr`专用的快捷聊天工具，可以方便你打`ygg`,`tks`等。    

## 功能   
- 快捷消息：      
	1. ygg pls!(5次）        
	2. Thank you!(3次)      
	3. 自定义。（内容和发送次数）     
- 鼠标位置自动回复（以很短的时间发完消息后鼠标回到原位）。
	> 注：发送消息时移动鼠标可能会导致出错。   
	> 作者正在努力改进。        
	> 欢迎你给出建议。

## 适用系统   
- Windows   

## 操作步骤    
1.	运行`ChatBomb-florr.exe`( *可能需要管理员权限，正常情况下不需要* )   
2.	按照提示语操作即可！
