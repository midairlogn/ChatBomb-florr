# ChatBomb-florr
这是`florr`专用的快捷聊天工具，可以方便你发`ygg`,`tks`等。    

## 功能   
- 快捷消息：  
	0. 重新设定程序参数。    
		1. ygg pls!(5次）        
	2. Thank you!(2次)         
	3. /squad-leave （1次） 
	4. Sorry . I don't have ygg!    
	5. 自定义。（内容和发送次数）         
- 鼠标位置自动回复（以很短的时间发完消息后鼠标回到原位）。  
	> 注：发送消息时幅度较大的移动鼠标或者按左右键或按键盘可能会导致出错，消息无法发出。     
	> 作者正在努力改进。            
	> 欢迎你给出宝贵的意见和建议。      

## 适用系统   
- Windows   

## 操作步骤    
1.	运行`ChatBomb-florr.exe`( *可能需要管理员权限，正常情况下不需要* )   
2.	按照提示语操作即可！
